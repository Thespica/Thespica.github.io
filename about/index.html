关于本站

站名

Spica ，中文名「角宿一」，室女座 $\alpha$ 星。北半球春季的夜晚，你能在东南的夜空看见它淡蓝的身影。我喜欢这颗星星，因此以 The Spica 命名本站。

建站

始建于 2023 年初。前端基于 Hexo 框架，主题为 Even；项目托管在 GitHub；由 Netlify 提供静态网站部署；Cloudflare 提供 CDN 加速。

2023-9-23：八月域名到期，出于经济和安全考虑不愿再续。将静态网页发布到 gh-pages 分支，并通过 GitHub Pages 部署，因此域名也迁移为 thespica.github.io。
